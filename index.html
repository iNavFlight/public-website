iNav

GPS aware flight controller

  • RCGroups
  • Github

iNav is designed to run on various flight controllers based on STM32 microcontrollers

  • Naze32
  • Flip32
  • CC3D
  • SPRacingF3
  • others

iNav utilizes various sensors to archive best performance

  • Gyroscope
  • Accelerometer
  • GPS
  • Barometer
  • Magnetometer
  • Ultrasonic rangefinder

Flight modes that suits your needs

  • Acro
  • Angle
  • Position Hold
  • Return To Home
  • Waypoints

Not only for quadcopters

iNav supports Tri, Hexa, Octa and Fixed Wing UAV platforms as well

When you do not need GPS

You can use iNav on 250 class or smaller racer as well
